##setting up the project
clone or fork this project or download the zip
it does not contain a build for executable, that means you need to rebuild when running

##Requirement
-marven must be installed
-and java jdk, intelij with build in tools preferebly
#Dependencies
--check pom.xml file and run the marven sync button on your top right corner
this pom file is inside the loot of this project

#Running
--this project uses java web sockets
for the client side you need to setup the proxy server on your machine,
on windows go to settings->network & internet->setup proxy server
on port use 8080 or 443, on ip use 127.0.0.1 then turn it on and save.
--this project uses http only not https so for testing use the site below or any site running http
http://httpbin.org/
